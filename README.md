# Evyatar Hazan Portfolio

Personal portfolio site for [Evyatar Hazan](https://github.com/Evyatar-Hazan), deployed with GitHub Pages.

## Stack

- React 19
- TypeScript
- Vite
- Framer Motion
- i18next
- GitHub Actions Pages deployment

## Local Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run validate
```

This runs linting, tests, and a production build.
